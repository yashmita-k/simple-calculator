# Simple Calculator

A clean, responsive, and minimalist calculator web application built from scratch using pure HTML, CSS (Grid & Flexbox), and JavaScript. 

## Features

* **Sleek Dark UI:** Designed with an elegant, modern dark theme that is easy on the eyes.
* **Perfect Alignment:** Uses **CSS Grid** with smart fractional units (`1fr`) and spanning layouts to ensure no empty diagonal spaces or misalignments.
* **Responsive Centering:** Built using **CSS Flexbox** to automatically center the calculator perfectly on any screen size.
---

## Tech Stack

* **HTML5:** Semantic layout and structure.
* **CSS3:** Advanced layouts (Flexbox, Grid) and responsive design.
* **JavaScript:** Dynamic UI updating, string expression parsing, and error handling.
